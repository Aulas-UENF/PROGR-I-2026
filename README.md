# 👋 Bem-vindos ao Repositório Central da Disciplina: Programação I

Olá, futuro(a) desenvolvedor(a)! 🚀

É uma alegria receber você nesta jornada. A disciplina de **PROGRAMAÇÃO-I** é a sua porta de entrada para o mundo da computação. Aqui, você não apenas aprenderá a lógica de programação, mas também as ferramentas que os profissionais usam todos os dias ao redor do mundo.

Neste repositório você encontrará todo o material necessário e os links para as atividades práticas.

### 🎯 Nosso Objetivo
Nesta disciplina, usaremos o **GitHub** como nossa sala de aula virtual e portfólio. Ao final do semestre, você terá um histórico real de tudo o que produziu, pronto para ser mostrado ao mercado.

### 🧭 Como se orientar por aqui:
- **Tudo centralizado:** Este repositório é o seu "mapa". Se estiver perdido, volte sempre para cá.
- **Materiais:** Os slides e PDFs estão na pasta `/materiais`.
- **Atividades:** Cada semana terá um link na tabela "Cronograma e Atividades". Ao clicar, o **GitHub Classroom** criará uma cópia particular para você fazer o exercício.
- **Dúvidas:** Confira primeiro as Dúvidas Frequentes [FAQ.md](./FAQ.md). Se o erro persistir, abra uma *Issue* ou entre em contato!
